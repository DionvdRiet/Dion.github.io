Artificial Creatures

Het belangrijkste wat een Easy Order automatic ordering machine nodig heeft is klanten. Zonder klanten kan het apparaat überhaupt geen functie uitvoeren. Wanneer een klant de McDonalds binnenstapt, wordt hij begroet door de machine. Nu de aandacht gevestigd is op de machine, raakt de klant de machine aan om het menu te bekijken, waardoor het apparaat een blij gevoel krijgt. Dankzij het touchscreen, wat ook erg belangrijk is voor het gebruik ervan, kan de klant beginnen met het plaatsen van een bestelling. Mocht de touchscreen niet werken, dan gaat de klant naar een ander machine, waardoor de machine zich verdrietig gaat voelen. Tijdens het bestellen ‘praat’ de machine met de klant door aanbevelingen te doen, zoals het erbij nemen van wat drinken of het nemen van een menu. De machine voelt zich hier erg goed bij, want hij kan de klant blij maken met een betere keuze. Uiteindelijk wanneer de klant zijn bestelling compleet heeft, kan de machine zijn bestelling versturen naar de medewerkers en de bon uitprinten voor de klant. De machine is blij dat hij wederom succesvol een klant heeft geholpen, op naar de volgende klant!
