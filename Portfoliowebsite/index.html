Over mij

Ik ben Dion van der Riet, een 20-jarig student Communication & Multimedia Design aan de Haagse Hogeschool. In de afgelopen twee jaar heb ik tijdens mijn opleiding veel ervaring opgedaan in de vorm van prototyping, programmeren (HTML, CSS, Javascript), user testing en voornamelijk concepting. Zo'n 5 jaar geleden ben ik begonnen met ontwerpen van banners voor o.a. YouTubers, maar sinds het begin van mijn studie is mijn liefde voor webdesign gaan groeien. Dankzij het programma Adobe Xd zet ik ideeën snel om in pixels. Omdat ik al een aantal jaren bezig ben met designen, vind ik het makkelijk om individueel een ontwerp uit te werken, maar met de vele groepsprojecten bij deze opleiding ben ik er toch wel achtergekomen dat klasgenoten ook hele fijne mensen zijn. Mijn favoriete projecten zijn projecten waar ik mijn creativiteit los kan laten op concepten, designs en code.

Case Studies

Gaming Application

Voordat ik aan dit project begon bestond er nog geen applicatie of website die een breed overzicht gaf van games die eraan zitten te komen. Binnen dit project heb ik het gehele designproces van onderzoek tot realisatie doorlopen.

Onderzoek / User Testing / Ontwerpen (Adobe XD) / Prototyping / HTML/CSS/Javascript

Tijd voor mars

In het jaar 2032 gaan astronauten naar Mars toe. De astronauten hebben behoefte aan het hebben van een klok die de huidige tijd op aarde weergeeft. Binnen dit project is gewerkt met HTML, CSS voor het ontwerp en Javascript voor o.a. het klokmechanisme. Daarnaast is Javascript gebruikt om de achtergrond van elke klok aan te laten passen op de huidige tijd. Om 6:00 is er een roze achtergrond, terwijl 20:00 de vallende avond moet weergeven.

Ontwerpen (Photoshop) / HTML/CSS/Javascript

Noothulp

Kinderen die op muziekles zitten hebben vaak maar eens per week muziekles. Dit betekent in de meeste gevallen dat een kind slechts één keer per week oefent met haar piano of gitaar. Nu de meeste kinderen toch een telefoon tot hun beschikking hebben, moet er een applicatie worden ontwikkeld die zonder dat een professional in de buurt hoeft te zijn feedback geeft over het bespelen van een instrument.

User Testing / Ontwerpen / Prototyping
